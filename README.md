# Krell Source

Krell is the Boldport Club version of MadLab's Noise-X noise synthesiser kit. See:

* [Noise-X ](http://www.madlab.org/kits/noise-x.html) - the original kit
* [Krell](https://www.boldport.com/products/krell/) - Boldport kit
